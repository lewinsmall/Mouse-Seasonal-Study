# Overview

This repository contains the code and processed data for the analysis of liver RNA sequencing data and tissue qPCR data contained in the following paper (https://pubmed.ncbi.nlm.nih.gov/37689069/). 

The "dataDGE.groups.Rdata" file contains count data for the C57BL/6N experiment, while the "Pooled Metadata.csv" contains the experimental design for this experiment. 

The "C3H.data.raw.Rdata" contains count data for the C3H experiment and the "C3H_Metadata_Sheet.csv" contains the experimental design for this experiment. 

The analysis script is contained in the R Script.R file. All other information about study design is provided in the linked publication.
